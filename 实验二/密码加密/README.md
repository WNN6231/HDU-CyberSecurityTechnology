# 综合加密系统

本系统是一个基于Python的综合加密解决方案，支持文本、图片、视频和结构化数据的加密与解密，提供了友好的图形用户界面。

## 功能特点

- **多种加密算法支持**：
  - 对称加密：SM4（国产算法）、AES、3DES
  - 非对称加密：RSA、ECC
  
- **多种数据类型加密**：
  - 文本加密
  - 图片文件加密
  - 视频文件加密
  - 结构化数据加密（Excel、CSV）
  
- **结构化数据加密模式**：
  - 全表加密
  - 列加密
  - 字段加密
  
- **完善的密钥管理**：
  - 密钥生成
  - 密钥导入/导出
  - 密钥存储与管理

## 系统要求

- Python 3.7+
- 依赖库：见 requirements.txt

## 安装步骤

1. 安装依赖库：

```bash
pip install -r requirements.txt
```

2. 运行主程序：

```bash
python main.py
```

## 使用说明

### 文本加密

1. 选择"文本加密"标签页
2. 选择加密类型（对称/非对称）
3. 选择加密算法和密钥
4. 输入要加密的文本
5. 点击"加密"按钮
6. 加密结果将显示在输出文本框中

### 文件加密

1. 选择"文件加密"标签页
2. 点击"浏览"选择要加密的文件
3. 选择加密类型（对称/非对称）
4. 选择加密算法和密钥
5. 点击"加密文件"按钮
6. 选择保存加密文件的位置

### 结构化数据加密

1. 选择"结构化数据加密"标签页
2. 点击"浏览"选择要加密的数据文件（Excel或CSV）
3. 选择加密模式（全表/列/字段）
4. 如果选择列或字段加密，选择要加密的列
5. 选择加密算法和密钥
6. 点击"加密数据"按钮
7. 选择保存加密数据的位置

### 密钥管理

1. 选择"密钥管理"标签页
2. 生成新密钥：
   - 选择密钥类型（对称/非对称）
   - 输入密钥名称
   - 选择算法和密钥长度
   - 点击"生成密钥"按钮
3. 导入密钥：
   - 输入密钥名称
   - 点击"导入密钥"按钮
   - 选择密钥文件
4. 导出密钥：
   - 在密钥列表中选择要导出的密钥
   - 点击"导出密钥"按钮
   - 选择保存位置
5. 删除密钥：
   - 在密钥列表中选择要删除的密钥
   - 点击"删除密钥"按钮

## 安全注意事项

1. 密钥安全：
   - 妥善保管密钥，避免泄露
   - 定期更换密钥
   - 对重要数据使用强密钥

2. 数据备份：
   - 加密前备份原始数据
   - 安全存储密钥，避免密钥丢失导致数据无法恢复

3. 算法选择：
   - 对敏感数据优先使用国产密码算法（SM4）
   - 根据安全需求选择合适的密钥长度

## 系统架构

- `main.py`：主程序，包含GUI界面
- `crypto_utils/`：加密工具模块
  - `symmetric.py`：对称加密实现
  - `asymmetric.py`：非对称加密实现
  - `key_manager.py`：密钥管理
  - `file_crypto.py`：文件加密
  - `structured_data.py`：结构化数据加密
